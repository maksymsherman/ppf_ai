# AI coding PPF

This directory contains a Python/uv chart for the AI-assisted coding production
possibilities frontier discussed in the prompt.

Generate the chart with:

```sh
uv run ai_coding_ppf.py
```

The script writes:

- `ai_coding_ppf.svg`
- `ai_coding_ppf.png`

The values are schematic indexes, not measured empirical data. The geometry is
the claim: AI moves the volume frontier much farther than the quality ceiling,
and the high-volume tail can keep pushing quality lower.
